## 💻Sobre
<p align="center">
    <img src="assets/img_IGTI.png" style="width: 10px">

## 📩Como baixar o projeto

```bash

 # Clonar o repositório
 $ git clone https://github.com/MarlonFerreira99/colorPicker

 # Entrar no diretório
 $ cd colorPicker

 # Instalar as dependências
 $ yarm install

 # Iniciar o projeto
 $ yarm start
```
